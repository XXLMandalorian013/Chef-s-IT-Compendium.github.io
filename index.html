  • Had fun dabbling with Docker and spun up a Pii cloud scanner tool for work. This required a Windows Server (2022), and in my case, as a VM, Windows Subsystem for Linux (WSL2), Ubuntu more specifically and I picked Windows Terminal to access and manage the ubuntu VM.

    I am a big fan of Terminal but had not tried to use it on a Windows Server and learned a few things like the Windows Store not being supported on Windows Server 2019 and higher, as well as WinGet only being accessible to Windows 10/11. So, I decided I would create a PowerShell script to download/install it manually! This was a fun little project as I had not delt with downloading/installing anything that used APPX, a file format that is used to package Windows applications. You may also know AppX as (RAD), Rapid Application Development. The only downfall with manually installing Terminal is that it does not get automatically updated.

    That said, I tried to create my install script as cookie cutter as possible so if anyone would only need to change a variable or two in each function to install the latest version as they are released. It was a bit easier for the uninstall script as it should not need to have anything changed to removed Terminal regardless of the version.
